# 🎓 Student Management System (Core Java)

A console-based application built using **Core Java** to perform basic CRUD operations on student records.

## 🌟 Key Features
* **Add Students:** Capture ID, Name, and Course details.
* **View Records:** Display all registered students in a clean list format.
* **Dynamic Storage:** Uses `ArrayList` for flexible data management.

## 🛠️ Concepts Used
* **Object-Oriented Programming (OOP):** Encapsulation using the `Student` class.
* **Java Collections:** Implementation of `ArrayList`.
* **Scanner Class:** For interactive user input.
* **Control Flow:** `while` loops and `switch/if-else` logic.

## 🚀 How to Run
1. Ensure you have **JDK** installed on your system.
2. Download `StudentManagement.java`.
3. Open your terminal/command prompt.
4. Compile: `javac StudentManagement.java`
5. Run: `java StudentManagement`
